# Holiday Card 2016

http://2-si.de/xmas/2016

## Install

To compress the images, you'll need a newish version of libpng. If you're on
macOS, this is arguably easiest via [Homebrew](http://brew.sh): 

```sh
$ brew install libpng
```

Then, it's just the normal npm silliness:

```sh
$ npm install
```

## Running

```sh
$ npm run dev
```
